# n8n-ai-workflow
# AI Email Generator Workflow (n8n)

This project demonstrates a simple n8n workflow integrating an LLM API.

## Features

* Manual trigger workflow
* Dynamic input handling
* LLM API integration
* Automated email generation

## How it works

1. Input text is provided
2. Workflow sends request to LLM API
3. AI generates a professional email

## File

* workflow.json (import into n8n)
